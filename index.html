Our Wedding

To add a bit of a personal touch on our big day, I decided to customize and host a website with details for our wedding on my own. I started with a design template from StyleShout, as I admit I am not the best when it comes to coming up with a great looking design. From there I customized the HTML and CSS to look how we wanted it and so it was easy for our guests to navigate and understand. JQuery was used to make the RSVP form dyanmic and it was connected to Mailchimp to keep track of the responses. The site was hosted using a Digital Ocean cloud server in which I spun up a Node/Express instance to run on. This was a fun project that was well received by all of our guests.

HTML, CSS, JavaScript, Node.js, Express.js, JQuery, Mailchimp, Digital Ocean

MyPantry

MyPantry was my final project when finishing up coding school. This was an independent project I completed from start to finish and proved to be the start of my portfolio. The purpose of the site was to update your digital pantry with ingredients you currently have at your house and search the internet for recipes you can make with what you already have. I used Materialize CSS and JQuery to help with the design and flow of the website. Like many of my other works, I used Node/Express to run the application on a Digital Ocean cloud server. The login and sign up feature used Passport.js for encryption of information and store user data and their ingredients in a MongoDB database. Lastly, the application connected to the Yummly API and used their database of over 1 million recipes to complete the searches in.

HTML, CSS, Matrialize CSS, JQuery, Node.js, Express.js, Passport.js, MongoDB, Yummly API, Digital Ocean
